Python vs. Scratch

Similarities

  1. One can use loops and use user inputs
  2. Operators (<, >, =) work the same
  3. Variables can be used in both languages

Differences

  1.Python has strings (str) and integers (int)
  2. In Python, you can make a new window pop up
  3. With Scratch, you can change how a character looks like with costumes

Feelings About Debugging

I had a partner (Lucia) and she would catch mistakes that I missed. That lessened the amount of debugging that I had to do. Having her made programming easier.
At times, we ran into bugs that required the help of a teacher. When we had to ask for help, I learned new information. Learning makes me happy because I know I am
becoming smarter. Debugging can be frustrating when the problem is not figured out, but when things do work out I feel very good.
